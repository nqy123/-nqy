# -nqy
对于吴恩达机器学习课程的作业进行编码和解释
其中包括线性回归，线性的逻辑回归，非线性的逻辑回归，svm，kmeans等
每个问题都从多个模型去解释和建模，包括随机森林，决策树，以及sklearns中的一些高级的API
